[![Status: En Desarrollo](https://img.shields.io/badge/Status-En%20Desarrollo-yellow)]()

**Micro-ERP Financiero** — Aplicación para gestionar tu vida financiera personal, incluyendo préstamos, ingresos, gastos, presupuestos y soporte multimoneda, con entrada de datos mediante lenguaje natural (NLP).

---

## Stack Tecnológico

| Capa | Tecnología |
|------|-----------|
| **Frontend** | Web (PWA), iOS, Android |
| **Hosting Frontend** | Vercel (CDN gratuito) |
| **Backend** | Docker sobre Oracle Linux |
| **Hosting Backend** | Oracle Cloud (Siempre Gratis) |
| **Base de Datos** | Oracle Database |
| **Autenticación** | JWT |
| **NLP / IA** | Google Gemini API |

---

## Funcionalidades Clave

- **NLP:** Registra gastos, ingresos y préstamos escribiendo en lenguaje natural.
- **Gestión de Préstamos:** Ciclo de vida completo con recálculo automático y comisiones.
- **Finanzas Personales:** Ingresos, gastos, presupuestos y contabilidad de partida doble oculta.
- **Dashboards:** Flujo de caja, proyecciones y rendimiento de cartera.
- **Multimoneda:** Soporte para múltiples divisas con tasas de cambio históricas.
