# Código de Conducta

## Nuestro Compromiso

Con el objetivo de fomentar un entorno abierto y acogedor, nos comprometemos a hacer de la participación en este proyecto una experiencia libre de acoso para todos.

## Estándares

Ejemplos de comportamiento que contribuyen a un entorno positivo:

- Usar lenguaje acogedor e inclusivo.
- Respetar puntos de vista y experiencias diferentes.
- Aceptar críticas constructivas.

Ejemplos de comportamiento inaceptable:

- Uso de lenguaje o imágenes sexualizadas.
- Comentarios insultantes o despectivos.
- Acoso público o privado.

## Aplicación

Los incidentes de comportamiento abusivo pueden ser reportados al maintainer del proyecto. Todas las quejas serán revisadas e investigadas.

## Atribución

Este Código de Conducta está adaptado del [Contributor Covenant](https://www.contributor-covenant.org), versión 2.1.
